## GET
- 서버에게 리소스를 요청하기 위해 사용
- HTTP/1.1은 서버가 이 메서드를 구현할 것을 요청한다.
- 요청 메시지 예제
```HTTP
GET /seasonal/index-fall.html HTTP/1.1
Host: www/example.com
Accept: *
```
- 응답 메시지 예제
```HTTP
HTTP/1.1 200 OK
Content-Type: text/html
Content=Length: 617

<HTML>
<HEAD>...
...
```
