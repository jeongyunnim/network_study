## 리다이렉션
- 웹 서버가 성공 메시지 대신 리다이렉션 응답을 반환한다.
- 리다이렉션 응답은 3XX로 지칭된다. Location 응답 헤더는 콘텐츠의 새로운 혹은 선호하는 위치에 대한 URI를 포함한다.
### 사용하는 경우
- 영구히 리소스가 옮겨진 경우 
	- 301 Moved Permanently
- 임시로 리소스가 옮겨진 경우
	- 303 See Other
	- 307 Temporary Redirect
- 기타
