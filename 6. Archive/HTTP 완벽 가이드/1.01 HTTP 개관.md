## HTTP는 웹 트래픽을 어떻게 전송하는가?
- [[트래픽]]
- 가장 흔한 클라이언트는 웹 브라우저(크롬, 파이어폭스 등)이다.
	- \http://www.naver.com/index.html 페이지에 접속하려고 한다면
		1. 브라우저는 HTTP 요청을 서버에 보낸다.
		2. 서버는 요청을 받은 객체(index.html)를 찾는다.
		3. 있다면 타입과 길이 등의 정보를 HTTP 응답으로 보낸다
